Group #1: FIRST (Devin Holland, Josh Lindoo, Tim Starr)


The database will contain information for each of the sections of this project. The database for the FIRST location will need a table that will log the students at Kettering who are part of FIRST teams and it will also need to keep track of the FIRST teams who are choosing to use the space. This would be through linking the table of FIRST teams to a table of usages of the space (time, date, tools etc.). We would want to build a UI where the user can search for a student and find out if they are part of the team. The FIRST space could be accessed by an ID card and we can set privileges for students (when they can access, etc).

Since this space will be used by many people it would be beneficial for a way to communicate between the people using the space. The Forum database would allow user (user table) to talk to one another through messages and boards (message/thread tables) and this would allow for people to collaborate on projects going on in the space.

The last part of this project is a Proposal system. With all of the things going on in the space people might want there to be things added to the space that would help their FIRST team build their robot easier or better. We would want to set up a proposal type table that would allow the user to specify the type of proposal (monetary, tool, etc.) and then give a description about what they want. Each proposal would be tied to a user that submitted the proposal. We would have someone be able to review and approve or deny the proposal but limit the people who can review down to people with privileges to do so.
