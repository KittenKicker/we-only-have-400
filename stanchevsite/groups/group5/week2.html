Group #5: WKUF Site(Devin Holland, Lucas Brandt, Jacob Miller)


Description

This project will be enhancements to Kettering’s radio station’s (WKUF’s) website. We will be adding features that will allow a user to submit a request for a specific song to be played without needing to call in to the radio station. Another feature that will be added is an on-demand radio section that will allow the user to choose a DJ that they would like to listen to and then pick one of their previous broadcasts to listen to online.

Need

Currently, WKUF does not have an effective way for a user to request a song. They would either have to know the phone number for the station or they would need to know the DJ on Facebook and contact them through that means. This is not an ideal way and it would be a big improvement for both the DJs and the people who wish to request a song to be played on air for them.

Approach

We will be starting with the on-demand and song request features. We will split up the team such that 2 people are working on the on-demand feature as it will require more work, and one member of the team will be working on the song request feature. If either section is finished first, the member(s) working on that section will help the other member(s) finish their section. If there is time after these features are added, then we will begin to improve existing areas of WKUF’s website.

Benefit

As stated previously, this project will greatly benefit the DJs of the radio station as well as Kettering students and members of the Flint community that listen to the station. This will consolidate the song request process into one location for the DJs to check and listeners to visit. Many other radio stations have a song request section. Some DJs of other radio stations have their broadcasts saved as podcasts for users to listen to from the website. This will help Kettering’s WKUF to continue to thrive and grow in listenership.

Competition

The competition to this project is other radio stations throughout the Flint area. These radio stations are commercial so it is not expected that our project will increase listenership to surpass any of these stations, but WKUF has a loyal listenership for its shows and this will help to make those listeners happy and hopefully attract more.
